## Install

- AppStore (recommend)

  > [**ARuler** https://itunes.apple.com/us/app/id1287068680](https://itunes.apple.com/us/app/id1287068680)


- Manually compile
  1.  `git clone https://github.com/TBXark/Ruler `
  2. `pod install`
  3. Compile project with Xcode
